# 🎬 Site de Cinema - Gran Turismo

Este projeto foi desenvolvido como atividade acadêmica com o objetivo de criar um site utilizando *apenas HTML*, seguindo requisitos específicos de estrutura e formatação.

---

## 📖 Sobre o projeto

O site tem como tema o cinema e apresenta o filme *Gran Turismo: De Jogador a Corredor*, baseado em uma história real inspiradora.

A proposta do trabalho inclui:

* Criação de *duas páginas HTML*
* Navegação entre páginas
* Uso de *imagens e textos*
* Estilização utilizando o atributo style diretamente nas tags

---

## 🧩 Estrutura do projeto

📁 Arquivos do projeto:

* index.html → Página inicial com introdução ao filme
* pagina2.html → Página com mais detalhes sobre o filme

---

## 🎯 Funcionalidades

* Navegação entre páginas com links
* Exibição de imagens relacionadas ao filme
* Conteúdo informativo sobre a história
* Layout estilizado com CSS inline

---

## 🚀 Tecnologias utilizadas

* HTML5

---

## 🎓 Objetivo acadêmico

Este projeto foi desenvolvido para praticar:

* Estruturação de páginas web
* Uso de tags HTML
* Criação de links entre páginas
* Aplicação de estilos inline

---

## 📸 Tema abordado

O filme mostra a jornada de um jogador de videogame que se torna piloto profissional, destacando temas como:

* Superação
* Determinação
* Transformação de sonhos em realidade
